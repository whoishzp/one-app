# ONE 更新日志

## v2.0.5 — 2026-07-09

### 新增
- **背景壁纸轮播**：支持多图轮播替换原单图背景，过渡丝滑如 macOS 桌面
- **内置 + 自定义双组壁纸**：9 张精选内置壁纸 + 自定义目录导入，点击缩略图一键切换
- **轮播设置**：间隔时间（30s~1h）、过渡效果（淡入淡出/直切）、播放顺序（顺序/随机）
- **自定义壁纸导入**：选择目录后自动压缩复制，源文件移动删除不影响使用

### 改进
- **默认背景图更新**：更换为新的默认背景图片

## v1.0.1 — 2026-06-29

首个公开发行版本。

### 新功能
- **AI 持续交互**：通过 CursorGood MCP，AI Agent 可以在工作过程中持续向你汇报进度、提问并等待回复
- **OneMind 知识管理**：三层结构（知识库 → 文件夹 → 文档），所见即所得编辑器，AI 可直接读写知识库
- **定时提醒**：循环/定点/一次性触发，全屏蒙层打断，8 套主题，下班模式
- **Soldier 远程助手**：通过飞书接收消息，调度 AI 执行任务
- **Fe 开发工具**：JSON 美化/比对、编码转换、时间戳转换
- **MCP 统一架构**：单 Server 支持多模块工具分发，`模块.动作` 命名空间
- **GitHub 云同步**：应用数据自动/手动同步到 GitHub，跨设备恢复
- **同步后自动重启**：拉取新数据后提示重启以应用配置
- **自定义主题**：4 套主题色 + 自定义配色、背景图 + 透明度调节
- **自动更新检测**：启动时自动检查新版本

### 修复
- OneMind 文档快速切换时数据丢失
- GitHub 同步远端领先时 push 失败
- OneMind 索引文件 Git 合并冲突
