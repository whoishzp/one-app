# OneMind

OneMind 是 ONE 内置的知识管理模块，采用「知识库 → 文件夹 → 文档」三层结构组织内容，配合所见即所得的富文本编辑器，打造个人知识体系。

## 知识库

知识库是最顶层的组织单元，类似"笔记本"概念。

- 在 OneMind 根节点下创建知识库
- 每个知识库有独立的首页，可编辑简介、查看文档统计
- 知识库内可自由创建文件夹和文档，层级不限

## 目录树

左侧目录树管理所有内容：

- **文件夹 + 文档**：树状组织，支持多层嵌套
- **新建**：右键或按钮新建文件夹/文档
- **重命名**：双击节点名称编辑，失焦自动保存
- **拖动排序**：支持拖动调整顺序和跨文件夹移动
- **复制 / 删除**：右键菜单操作
- **自定义图标**：点击节点图标可弹出图标选择器，支持 360+ Unicode Emoji 和 250 个内置图标

### 图标选择器

三个 Tab 页：

| Tab | 内容 |
|---|---|
| 推荐 | 每类精选 emoji + 常用图标 |
| 表情符号 | 9 大类别 360+ Unicode emoji |
| 图标 | 250 个内置精灵图标 |

选中即生效，也可一键恢复默认图标。

## 文档编辑器

基于 Tiptap 的所见即所得富文本编辑器，支持多种内容格式：

### 工具栏

编辑器顶部内联工具栏，格式操作即时反馈：

- 文字格式：加粗、斜体、删除线、行内代码
- 标题：H1 / H2 / H3
- 列表：无序列表、有序列表、待办列表
- 引用、分割线、代码块
- 字体颜色：40 色调色板
- 背景高亮：40 色高亮面板
- 文档大纲：右侧浮层展示 H1/H2/H3 目录，点击跳转

### Slash 命令

输入 `/` 调出命令菜单，快速插入各种块级元素。菜单由插件动态驱动，新增 Block 类型时自动出现在菜单中。

### 图片

支持粘贴和拖拽插入图片，文件自动保存到本地。

### 主题适配

编辑器自动跟随系统深色/浅色模式切换，无闪烁。

## 文件类型扩展

OneMind 采用插件化的文件类型系统，注册新的文件类型后目录树 UI 自动适配，无需修改其他代码。
