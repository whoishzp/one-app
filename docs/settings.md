# 系统设置

ONE 的全局配置中心，管理应用外观、行为和数据同步。

## 主题风格

4 套预设主题色 + 1 套自定义配色，自动适配系统浅色/深色模式：

| 主题 | 风格 |
|---|---|
| 中性灰 | 默认，低调百搭 |
| 冷蓝 | 清爽专注 |
| 森绿 | 自然舒适 |
| 暗紫 | 沉稳优雅 |
| 自定义 | 自由设置窗口底色和面板底色（浅/深色各一套） |

切换主题即时生效，所有模块自动刷新。

## 自定义背景图

- 全窗口背景图（覆盖导航栏和内容区）
- 透明度滑块实时调节
- 面板透明度独立调节，背景图可透过内容面板显示
- 背景图跟随 GitHub 同步

## 开机自启

开启后 ONE 随系统启动自动运行。

## 启动命令

应用启动时自动执行配置的 Shell 命令：

- 支持多条命令，各自独立启用/禁用
- 一键复制完整命令到剪贴板
- 一键手动执行

## AI Skill 管理

管理 Cursor AI 可调用的 Skill 文档，支持导出和安装。

## GitHub 云同步

所有配置数据同步到 GitHub 仓库，跨设备共享：

- **自动同步**：定时自动上传（默认 60 秒间隔）
- **手动上传**：一键推送当前配置
- **恢复**：从 GitHub 拉取配置并覆盖本地（二次确认）
- **SSH Key 检测**：自动检测 SSH 配置状态
