# 定时提醒

ONE 的核心模块，帮助你建立规律的工作节奏。支持多条规则并行、三种触发方式、全屏蒙层强制打断，确保提醒不会被忽略。

## 三种触发方式

### 循环提醒

每隔 N 分钟触发一次。适合番茄钟、定时休息等场景。

- 重启应用后自动恢复剩余倒计时，不会中断节奏
- 可搭配跟进提醒，关闭后 N 分钟再来一次

### 定点提醒

每天在指定时刻触发，可配置多个时刻。适合固定日程：站会、午休、下班。

### 一次提醒

指定某个时间点触发一次，触发后自动停用。适合临时事项。

## 提醒类型

每条规则可选择两种提醒方式：

| 类型 | 效果 |
|---|---|
| **桌面提醒** | 全屏蒙层覆盖所有屏幕，强制打断当前工作 |
| **定时脚本** | 按相同的时间策略执行 Shell 命令，适合自动化任务 |

定时脚本的执行日志会自动写入配置的日志目录，记录标准输出、标准错误和退出码。

## 全屏蒙层

弹窗覆盖所有屏幕（包括扩展屏），切换桌面后自动抢回焦点。

- **倒计时锁定**：可设置最短停留时长（例如强制看 30 秒），倒计时结束前无法关闭
- **Enter 后门**：紧急情况下可连续快速按 Enter 强制显示关闭按钮，次数可配（1~100，默认 4 次）
- **实时时钟**：蒙层中央显示当前时间，每秒刷新
- **ESC 关闭**：倒计时结束后支持 ESC 键快速关闭
- **自定义关闭按钮文案**：每条规则可自定义关闭按钮文字

### 8 套蒙层主题

| 主题 | 风格 |
|---|---|
| 黑白极简 | 纯净、无干扰 |
| 深蓝平静 | 冷静、放松 |
| 深红警告 | 紧迫、强提醒 |
| 深绿清新 | 自然、舒适 |
| 粉嫩可爱 | 活泼、轻松 |
| 马卡龙 | 柔和、甜美 |
| 冰霜冷调 | 清凉、克制 |
| 温柔晚霞 | 温暖、惬意 |

编辑规则时可实时预览各主题效果。

## 跟进提醒

任意触发模式均可开启跟进提醒：关闭蒙层后 N 分钟自动再弹一次，防止"看了就忘"。设为 0 表示不跟进。

## 实时状态面板

查看所有规则的运行状态：距下次触发的倒计时和进度条，一目了然。

## 醒目元素

每条规则可选择蒙层中央显示的内容：

- **时间**（默认）：大时钟居中
- **提示文字**：自定义大文字居中，时钟缩小到下方

# 下班模式

一键进入"已下班"状态，全屏黑幕遮住屏幕，暂停所有提醒。

## 功能

- 点击右上角「下班」按钮或使用全局快捷键进入
- 全屏黑幕覆盖所有屏幕，防止系统休眠
- 黑屏上有漂浮的动态眼睛动画，多个眼睛以物理碰撞方式自由移动
- 可选密码保护（在系统设置中配置），防止误退出
- 退出下班模式后自动恢复之前暂停的提醒规则

## 全局快捷键

在系统设置中可配置下班快捷键（需含修饰键），在任意应用中均可触发下班 / 取消下班。
