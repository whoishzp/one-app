# Fe 助手

Fe 助手是 ONE 内置的前端开发工具集，提供日常开发中高频使用的数据处理和转换功能。

## JSON 美化

将 JSON 字符串格式化为可交互的树形视图。

- **树形展开/折叠**：逐层展开 JSON 结构，快速定位目标字段
- **嵌套 JSON 自动解析**：字符串值中如果包含 JSON，自动识别并展开为子树
- **悬停复制**：鼠标悬停在任意节点上，可一键复制该节点及其所有子节点的 JSON
- **叶子节点框选**：叶子值支持鼠标框选复制
- **复制全部**：一键复制格式化后的完整 JSON

## JSON 比对

双面板 JSON 差异对比工具。

- 左右双面板，可直接编辑 JSON 内容
- 粘贴后自动格式化
- 差异行黄色高亮，快速定位区别

## 信息编码转换

支持 18 种常用编码/解码操作：

| 类别 | 操作 |
|---|---|
| 编码 | Unicode、URL、Base64、HTML 实体 |
| 加密 | MD5、SHA1 |
| 格式 | HEX 编解码 |
| 解析 | JWT 解码、Cookie 解析 |
| ... | 更多编解码操作 |

输入文本后选择操作类型，即时转换。

## 时间戳转换

时间与时间戳互转工具。

- **智能解析**：自动识别多种时间格式（时间戳、ISO 8601、常见日期格式等）
- **实时当前时间**：一键获取当前时间戳
- **快捷操作**：常用时间快捷按钮（今天开始/结束、昨天、本周等）
- **双击复制**：双击结果即可复制
